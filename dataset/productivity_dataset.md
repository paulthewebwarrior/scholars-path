# Dataset

This file contains 20,000 student records with detailed information about academic habits, lifestyle, digital distraction, and productivity levels.

Each row represents one student, and each column represents a specific feature influencing productivity.

Key columns include:

study_hours_per_day – Daily study time • sleep_hours – Daily sleep duration • phone_usage_hours – Daily phone usage • social_media_hours – Time spent on social media • gaming_hours – Daily gaming time • stress_level – Student stress level (1–10) • focus_score – Student focus score (0–100) • attendance_percentage – Class attendance percentage • final_grade – Academic performance grade • productivity_score – Target variable representing overall productivity (0–100)
